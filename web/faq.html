Frequently asked questions

Everything you might want to know about ShortLink.

Is ShortLink free to use?

Yes — completely free, with no sign-up and no account.

Do short links expire?

No. Links are stored persistently and keep redirecting to the original URL as long as the service runs.

Do you track or sell my data?

No. There's no account, no third-party trackers, and we don't sell data. Only the URL and its generated code are stored. See our Privacy Policy.

How are short codes generated?

Each URL gets a sequential ID, which is run through a reversible Feistel permutation and encoded as a 6-character base62 code. Codes are non-sequential and never collide.

Is there an API?

POST /encode and POST /decode accept and return JSON at api.short-link.fun. See the GitHub README for request/response details.
